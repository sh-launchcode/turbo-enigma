AIMprove game

A personal project written in React.js. The idea behind this game was to help improve the aim of
first person shooter gamers. As one myself, you can never get enough practice. Boxes will appear
all across the canvas. Your goal is to just click on them. The game itself isn't finished yet and I will
keep on improving it as I can.

Box Game

One of my first projects I made as a software developer. I made this game at ASU for honors credit,
and at the time of when I made it, it actually worked. I made this with Pygames, an API specific towards making python
games. You use the arrow keys to move around a window to cover other small boxes and improve your score. You have only 100 moves to cover as many boxes as you can!
